# Trilha de Java – Proway + Capgemini

Este repositório reúne todo o conteúdo que desenvolvi durante a trilha de Java, um curso fornecido pela Proway em parceria com a Capgemini.

O curso abrange conceitos de Java, Java Web, HTML, CSS, JavaScript, jQuery e Angular, e aqui você encontrará a minha evolução ao longo do aprendizado, incluindo exercícios, desafios e projetos práticos.

Conteúdo do repositório

HTML: do básico ao avançado

CSS: fundamentos, layouts e estilos avançados

JavaScript e jQuery: manipulação de DOM, eventos e interatividade

Java: conceitos iniciais, programação orientada a objetos e desenvolvimento web com Java

Observação: a parte de Angular será disponibilizada em outro repositório, devido ao tamanho e à extensão dos arquivos.

Objetivo deste repositório

O objetivo é documentar minha evolução e aprendizado, servindo como referência para estudos futuros e como portfólio de prática e projetos desenvolvidos ao longo do curso.
